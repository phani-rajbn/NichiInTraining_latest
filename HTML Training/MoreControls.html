Info about SOA

SOA means Service Oriented Architecture that we will learn in .NET Technology for developing web apps that talk to the back end data which could either a filesystem or database. We use AJAX to interact with the Services. AJAX is asynchronous Javascript and Xml.

Prior to HTML5, there were only 2 Tags to work like container Tags. div and p. But in real time, we needed different kinds of such divs for navigation, articles, header, footer, sections and many more. All were doing the same job as a containers. So div components were created as Classes(CSS) and based on the type we wer using the same div tags for different purposes.
Now in HTML5, we have tags called nav, article, header, footer, section, fieldset and many more...
