Contents of an HTML Page

  • Any html page will have a doctype that called as Prelude which tells that its an HTML5 Document.
  • It has 2 parts: head and body.
  • The head segment will contain the data(Meta information) about the Page like Author, Created Date, Content type, Source, Keywords for the search engine. The Head part also can contain the external links to this document like CSS files, Scripting files. It can also have title for the document.
  • The body segment is the content that U want to display on the page. This includes Text, Images, links, Tables, Forms that recieve inputs from the users, marques and many more.

Textual Formats for the body

Content in the text can be done in 2 ways. HTML 4.0 gave certain attrbutes and Tags to make the content formattable. Now its recommended to use CSS for applying the required formats. But still Web Apps use inline tags for formatting small amounts of content specific to that page.

The Important ones are:

  1. <b> Used for Bold text to draw the attention
  2. <i> Used for Italics text to emphasize the content
  3. <strike> Used for Strike text to strike the content
  4. <u> Used for underline text to draw the attention and mark for a significance
