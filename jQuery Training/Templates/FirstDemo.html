What is jQuery?

About jQuery

jQuery is a JavaScript Framework for developing JS applications in a simplified manner. It makes writing JS code easy. It provides a high level of abstraction for many operations which needs lot of code to be written in pure JS..It simplies the code for manipulating the DOM elemnts within the page using jQuery...

What is $

The shortCut for loading the jQuery is using $. It represents the library. It takes the arg of the selector on which U wish to perform operation.
Selectors will work like document.getElement... by Id, by TagName and ClassName...
